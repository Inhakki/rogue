Class: ResourceManager

ResourceManager

new ResourceManager()

Represents a manager that loads any CSS and Javascript Resources on the fly.

Methods

staticResourceManager.flush()

Removes all cached resources.

staticResourceManager.initialize()

Upon initialization.

staticResourceManager.loadCss(paths, callback)

Loads css files.

Name Type Description
paths Array | String

An array of css paths files to load
callback function optional

Function to be called when files are loaded

staticResourceManager.loadScript(paths, callback)

Loads a javascript file.

Name Type Description
paths string | Array

The path to the view's js file
callback function

Function to be called when script is loaded

staticResourceManager.unloadCss(paths, callback)

Unloads css paths.

Name Type Description
paths string | Array

The css paths to unload
callback function optional

Callback that fires when done

Name Type Description

The paths of the scripts to unload
callback function optional
