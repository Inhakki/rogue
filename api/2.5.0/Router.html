Class: Router

Router

new Router(options){Router}

Represents a manager that handles all routes throughout the app.

Name Type Description
options
Name Type Description
config String | Object

Configuration data or url to file that has it
el HTMLElement

The element to apply loading class to when loading a page

Methods

_fetchConfig()

Fetches the config.

getHistory(){Array}

Returns an array containing all urls that were hit previously.

Returns:
array of objects

getRelativeUrl(){string}

Gets the current relative url.

Returns:
a url string

getRelativeUrlParams(){Array}

Gets the current relative params.

Returns:
an array of params

goBack(index)

Navigates to previous url in session history.

Name Type Description
index Number

an index with a position relative to the current page (the current page being, of course, index 0)

goForward(index)

Navigates forward (if gone back).

Name Type Description
index Number

an index with a position relative to the current page

reset()

Resets Route Manager.

start()

Starts managing routes.

stop()

Stops routing urls.

triggerRoute(url, options)

Navigates to a supplied url.

Name Type Description
url string

The url to navigate to.
options Object optional

Set of navigation options

Name Type Description
trigger boolean optional

True if the route function should be called (defaults to true)
replace boolean optional

True to update the URL without creating an entry in the browser's history
