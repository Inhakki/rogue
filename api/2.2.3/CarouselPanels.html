Adds functionality for carousel panels.

Name Type Description
options object

Options passed into instance

Name Type Description
panels HTMLCollection

The panels in which to use for the carousel (an array of photos)
assetClass string optional

The CSS class of the asset images inside of the DOM
assetLoadingClass string optional

The CSS class that gets added to an asset when it is loading
autoLoadAssets boolean optional

Whether or not to automatically load assets when active
panelActiveClass string optional

The CSS class that gets added to an panel when it becomes active
onChange CarouselPanels~onChange optional

When the current panel is changed
lazyLoadAttr string optional

The attribute containing the url path to content that is to be lazy loaded
